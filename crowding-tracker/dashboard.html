What the number means. A station-level busyness index from TfL's anonymised, aggregated station WiFi data — a fraction of that station's busiest week since July 2019 (so not a headcount, and not comparable between stations). TfL's bands: <0.4 quiet · 0.4–0.7 busy · >0.7 very busy.

Live & typical read straight from api.tfl.gov.uk/crowding each time you open this page. The logged trend comes from history.csv, which a GitHub Action fills in every ~5 min during commute windows.
